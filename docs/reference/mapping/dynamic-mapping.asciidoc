[[dynamic-mapping]]
== 动态映射

Elasticsearch 最重要的功能之一是它试图摆脱你的方式，让你尽快地开始探索你的数据。
要索引文档，你不必先创建索引、定义映射类型和定义字段 -- 只需要索引文档，而索引、类型和字段将自动生效：

[source,js]
--------------------------------------------------
PUT data/_doc/1 <1>
{ "count": 5 }
--------------------------------------------------
// CONSOLE
<1> 创建 `data` 索引， `_doc` 映射类型、名为 `count` 类型为 `long` 的字段。

自动检测和添加新字段称为 _dynamic mapping_ 。可以自定义动态映射规则以满足你的目的：

<<dynamic-field-mapping,动态字段映射>>::

    管理动态字段检测的规则

<<dynamic-templates,动态模板>>::

    自定义规则，用于配置动态添加字段的映射。

提示：<<indices-templates,索引模板>>允许你配置新索引的默认映射，设置和别名，无论是自动创建还是显式创建。
