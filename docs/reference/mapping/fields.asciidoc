[[mapping-fields]]
== 元字段（Meta-Fields）

每个文档都有与之关联的元数据，例如 `_index` ，映射 <<mapping-type-field,`_type`>> 和 `_id` 元字段。创建映射类型时，可以自定义某些元字段的行为。

[float]
=== 身份元字段（Identity meta-fields）

[horizontal]
<<mapping-index-field,`_index`>>::

    文档所属的索引。

<<mapping-uid-field,`_uid`>>::

    由 `_type` 和 `_id` 组成的复合字段。

<<mapping-type-field,`_type`>>::

    文档的<<mapping-type,映射类型>>。

<<mapping-id-field,`_id`>>::

    文档的 ID。

[float]
=== Document source meta-fields

<<mapping-source-field,`_source`>>::

    表示文档正文的原始 JSON。

{plugins}/mapper-size.html[`_size`]::

    `_source` 字段的大小（以字节为单位），由 {plugins}/mapper-size.html [`mapper-size` 插件]提供。

[float]
=== Indexing meta-fields

<<mapping-all-field,`_all`>>::

    _catch-all_ 字段，用于索引所有其他字段的值。默认情况下禁用。

<<mapping-field-names-field,`_field_names`>>::

    文档中包含非空值的所有字段。

<<mapping-ignored-field,`_ignored`>>::

    文档中由于 <<ignore-malformed,`ignore_malformed`>> 而在索引时被忽略的所有字段。

[float]
=== Routing meta-field

<<mapping-routing-field,`_routing`>>::

    自定义路由值，用于将文档路由到特定分片。

[float]
=== Other meta-field

<<mapping-meta-field,`_meta`>>::

    Application specific metadata.
